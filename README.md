# QA API Testing - JSONPlaceholder

Proyecto de pruebas manuales de API utilizando Postman.

## Objetivo
Validar el funcionamiento de endpoints de la API pública JSONPlaceholder mediante pruebas funcionales manuales.

## API utilizada
https://jsonplaceholder.typicode.com/

## Herramientas
- Postman
- API REST
- JSON

## Endpoints probados

GET /posts  
GET /posts/{id}  
POST /posts  
GET /users/{id}

## Tipos de pruebas realizadas

- Validación de status codes
- Validación de estructura JSON
- Casos positivos
- Casos negativos
